**COURSE OVERVIEW:**

This course introduces the fundamental principles of data communication and computer networking that form the backbone of modern digital communication systems. It covers the evolution of the Internet, network architectures, protocols, standards, and various network topologies. Students learn about the OSI and TCP/IP reference models and their role in network communication. The course explains switching techniques, transmission media, signal representation, and channel capacity concepts at the physical layer. Data link layer topics such as line coding, framing, error detection and correction, and medium access control protocols are discussed in detail. Network layer concepts include IPv4 and IPv6 addressing, IP datagrams, ICMP, mobility management, and routing algorithms. The transport layer focuses on reliable data transfer mechanisms, TCP and UDP protocols, flow control, error control, and congestion control. Application layer services such as DNS, SMTP, TELNET, and SSH are also explored. The course provides both theoretical foundations and practical insights into the design and operation of computer networks. Upon completion, students will be able to analyze, configure, and troubleshoot networking systems used in modern communication environments.

**COURSE OBJECTIVE (S):**

The objectives of this course are to:

1\. To introduce the fundamental concepts of data communication, computer networks, layered architectures, switching techniques, and physical layer technologies.

2\. To develop an understanding of data link layer functions including coding techniques, framing, error detection and correction methods, and medium access control protocols.

3\. To provide knowledge of network layer operations, IP addressing schemes, routing algorithms, and next-generation Internet protocols such as IPv6.

4\. To familiarize students with transport and application layer protocols, enabling them to analyze reliable data communication, network services, and Internet-based applications
